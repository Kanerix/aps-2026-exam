# APS 2026 Exam

This repository contains our exam submission for the course in Algorithmic Problem Solving.

## Problems

We have tackled the following problems:

- **Buzzwords** — A string problem focused on finding the longest repeated
  substring within a given input.
- **Cookie Selection** — An array-based problem requiring efficient selection
  and ordering strategies.
- **Exchange Rates** — A dynamic programming problem involving optimal CAD/USD
  currency conversion, accounting for a commission fee on each trade.

## Our Own Problem

In addition to the selected problems, we designed and implemented our own
problem centered around **maximum flow**. The problem is set in the context of a
plumbing network, where two toilet waste pipes connect through a series of
manifolds to a sewer. The goal is to determine which of the two pipes can
discharge the most water, by computing the maximum flow from each source to the
shared sink.
